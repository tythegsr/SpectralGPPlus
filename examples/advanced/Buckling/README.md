# Examples
The two .py files in this folder have 10 different seeds run on the same models used in the jupyter notebook file, just for getting the average metrics of each model. Please go through the jupyter notebook first to understand the difference between the models.

Problem: Buckling - Multifidelity (1 HF 1 LF)

How to run: File(s) should run without errors by pressing run file. Feel free to adjust the seed, data generation, and model inputs such as the combined kernel and its encoders to test different setups.

Expected output: Model metrics, training time, and all latent embeddings + plots
